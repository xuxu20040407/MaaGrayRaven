# MaaGrayRaven

基于MaaFramework的 战双签到 小助手(V0.0)。图像技术 + 模拟控制，解放双手！  
由 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 强力驱动！

## 功能介绍

目前已有的功能：
进入游戏主页后：
- 进入宿舍，完成一键抚摸、派遣、购买特惠涂装
- 进入商店，购买逆元碎片
- 进入邮件，领取邮件奖励
- 进入采购，领取每日\周礼包

近期可能会支持的功能：

- 自动进入游戏界面
- 用血清刷去材料
